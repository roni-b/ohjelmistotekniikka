### Asennus

Step 1:
```bash
poetry install
```
Step 2:
```bash
poetry run invoke build
```
Step 3: 
```bash
poetry run invoke start
```
**Sovelluksen toiminta on testattu Pythonin versiolla 3.11.2 ja Poetryn versiolla 1.4.1**
